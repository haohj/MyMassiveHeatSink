# MyMassiveHeatSink

## 反熵热量中和器

此mod的作用是将缺氧游戏中的反熵热量中和器添加到研究栏和建造栏中，使得其可以被研究和建造出来，并且提供了一部分的参数可供配置。

修改调整部分：

+ 可以配置每秒的吸/产热量

  原版是每秒-16度，也就是工作时每秒吸热16度，设备工作自身还吸热64度，一共每秒吸热80度。现在可以调整工作时吸/产热量，设备自身吸热64度保持不变

+ 可以配置设备的最低工作温度

  原版最低温度是100K(也就是-173.15℃)，达到温度后就停止工作了

+ 设备不可被淹没

+ 设备不可被掩埋

## 目录结构

|-- BuildingFence.cs 将物品添加到建造栏中

|-- MassiveHeatSinkConfig.cs 物品的属性和输入输出元素配置

|-- Patch.cs 加载翻译配置

|-- STRINGS.cs 定义MOD名称和描述

|-- Study.cs 将物品添加到研究栏中

|-- Utils.cs 工具类，用于加载本地的翻译文件

## 安装指南

在Steam的创意工坊中搜索“反熵热量中和器”，然后订阅即可。
